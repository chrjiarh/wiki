    Mac 环境配置¶

    1. 安装 Command Line Tools¶

    • 方法一：使用 Terminal 运行如下代码安装，默认安装最新版本

      1
      xcode-select --install

      在弹出的窗口点击安装。

    • 方法二：登录苹果开发网站下载 .dmg 文件安装

      下载地址：https://developer.apple.com/download/all/

    验证安装，如果返回路径则表示安装成功。

    1
    xcode-select --print-path

    2. 关闭 MacOS 自动更新¶

    首先关闭所有自动更新和安装更新的按钮：系统设置 > 通用 > 软件更新 > 自动更新。

    然后配置屏蔽网络访问，让系统无法检测软件更新。

    1
    sudo vim /etc/hosts

    添加以下内容：

    1

    注意：使用系统代理或虚拟专用网时，将绕过本地 DNS 设置，所以仅设置 hosts 文件仍然会检测到更新。

    function proxy_off(){
    unset http_proxy
    unset https_proxy
    unset socks_proxy
    unset ftp_proxy
    unset rsync_proxy
    unset HTTP_PROXY
    unset HTTPS_PROXY
    unset SOCKS_PROXY
    unset FTP_PROXY
    unset RSYNC_PROXY
    echo -e "已关闭代理"
}
function proxy_on() {
    # 设置不需要代理的地址
    export no_proxy="localhost,127.0.0.1,localaddress,.localdomain.com"
    export http_proxy="http://127.0.0.1:7890"
    export https_proxy=$http_proxy
    export socks_proxy="socks5://127.0.0.1:7890"
    export ftp_proxy=$http_proxy
    export rsync_proxy=$http_proxy
    export HTTP_PROXY=$http_proxy
    export HTTPS_PROXY=$http_proxy
    export SOCKS_PROXY=$socks_proxy
    export FTP_PROXY=$http_proxy
    export RSYNC_PROXY=$http_proxy
    # echo -e "已开启代理"
}
function proxy_status() {
    echo $https_proxy
    curl cip.cc
}

    8. 安装 Docker 部署环境¶

    8.1 安装 OrbStack¶

    Orbstack 是一个基于 Docker 的轻量级虚拟化平台，专门针对 Mac 进行了优化，尤其是在 M 芯片的设备上。直接安装 Orbstack，而无需先安装 Docker。

    官网：https://orbstack.dev/。

    需要修改配置，否则 Docker 拉取不到镜像。

    修改配置文件：~/.orbstack/config/docker.json 
    1
2
3
4
5
6

    {
  "proxies" : {
    "http-proxy" : "http://127.0.0.1:7890",
    "no-proxy" : "localhost,127.0.0.1",
    "https-proxy" : "http://127.0.0.1:7890"
  }
}

    修改配置后，重新启动 Docker

    1
    orb restart docker
