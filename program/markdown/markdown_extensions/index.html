    Markdown 扩展语法¶

    • Python Markdown 提供了灵活的扩展机制。
    • PyMdown Extensions 包是一个优秀的附加扩展集合，非常适合高级技术写作。

    1. Abbreviations 缩写¶

    Abbreviations 缩写可以通过使用类似于URL和脚注的特殊语法来定义，以*开头，紧接着是方括号中要关联的术语或缩写：

    Text with abbreviations
    1
2
3
4
    The HTML specification is maintained by the W3C.

*[HTML]: Hyper Text Markup Language
*[W3C]: World Wide Web Consortium

    The HTML specification is maintained by the W3C.

    2. Admonition 警告¶

    Admonition 扩展添加了对警告的支持，通常称为标注，可以使用简单的语法在 Markdown 中定义。

    Admonition
    1
2
    !!! note "change title"
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.

    change title

    Lorem ipsum dolor sit amet, consectetur adipiscing elit.

    删除标题

    Admonition without title
    1
2
    !!! note ""
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.

    Lorem ipsum dolor sit amet, consectetur adipiscing elit.

    可折叠块

    Admonition, collapsible
    1
2
    ??? note
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
    Note

    Lorem ipsum dolor sit amet, consectetur adipiscing elit.

    在 ??? 后面添加 + 可折叠并自动展开

    Admonition, collapsible and initially expanded
    1
2
    ???+ note
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
    Note

    Lorem ipsum dolor sit amet, consectetur adipiscing elit.

    内联块

    Admonitions 警告也可以呈现为内联块（例如，对于侧边栏），使用 inline + end 修饰符将它们放置在右侧，或仅使用 inline 修饰符将它们放置在左侧

    Lorem ipsum

    Lorem ipsum dolor sit amet

    1

    !!! info inline end "Lorem ipsum"

    Lorem ipsum dolor sit amet

    6. Footnotes 脚注¶

    Footnotes 脚注是在不中断文档流程的情况下为特定单词、短语或句子添加补充或附加信息的好方法。

    Text with footnote references
    1
2
3
4
5
6
7
    Lorem ipsum[^1] dolor sit amet, consectetur adipiscing elit.[^2]

[^1]: Lorem ipsum dolor sit amet, consectetur adipiscing elit.
[^2]:
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla et euismod
    nulla. Curabitur feugiat, tortor non consequat finibus, justo purus auctor
    massa, nec semper lorem quam in massa.

    Lorem ipsum1 dolor sit amet, consectetur adipiscing elit.2

    7. Grids 网格¶

    Grids 网格非常适合构建索引页面，显示文档大部分内容的简要概述。

    8. Tables 表格¶

    Tables 可以在文本中插入数据表格，用于展示数据。通过添加自定义的 CSS 样式和 JS 脚本，可以实现排序、对齐等多种增强功能。

    9. Tooltips 工具提示¶

    Tooltips 为每个链接指定一个标题，当启用改进的工具提示时，该标题将呈现为漂亮的工具提示。

    Link with tooltip, inline syntax
    1
    [Hover me](https://example.com "I'm a tooltip!")

    Hover me

    还可以将工具提示添加到链接引用中

    Link with tooltip, reference syntax
    1
2
3
    [Hover me][example]

  [example]: https://example.com "I'm a tooltip!"

    Hover me

    10. Arithmatex 算术软件¶

    Arithmatex 扩展允许渲染块和内联块方程，并与 MathJax（数学排版库）无缝集成。
    MathJax 和 KaTeX 是两个流行的库，用于在浏览器中显示数学内容。

    11. Caret, Mark & Tilde 下划线, 高亮 & 删除线¶

    Caret、Mark 和 Tilde 扩展添加了使用简单语法突出显示文本并定义下标和上标的功能。

    Caret, Mark & Tilde
    1
2
3
    ^^下划线^^
==马克高亮==
~~删除线~~

    下划线
    马克高亮
    删除线

    12. Critic 评论¶

    Critic 扩展允许使用 Critic 标记来突出显示文档中添加、删除或更新的部分，即用于跟踪 Markdown 语法的更改。

                items[j], items[j + 1] = items[j + 1], items[j]

    15. Emoji 表情符号¶

    Emoji 表情符号扩展会自动将 *.svg 文件格式的捆绑和自定义图标以及表情符号内联到生成的 HTML 页面中。 Material for MkDocs中使用： Icons, Emojis

    17. Tasklist 任务列表¶

    Tasklist 启用任务列表后，无序列表项可以以 [ ] 为前缀来呈现未选中的复选框，或以 [x] 为前缀来呈现已选中的复选框，从而允许定义任务列表：

    Task list

2

- [ ] Vestibulum convallis sit amet nisi a tincidunt

    * [x] In scelerisque nibh non dolor mollis congue sed et metus
    * [ ] Praesent sed risus massa
